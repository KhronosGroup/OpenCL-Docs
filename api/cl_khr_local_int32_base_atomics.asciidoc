=== Description

`cl_khr_local_int32_base_atomics` allows atomic operations to be performed
on 32-bit signed and unsigned integers in local memory.

This extension became a core feature in OpenCL 1.1, with the built-in atomic
function names changed to use the **atomic_** prefix instead of **atom_**
and the `volatile` qualifier added to the pointer parameter _p_.
