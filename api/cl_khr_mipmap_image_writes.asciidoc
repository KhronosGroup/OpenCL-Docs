=== Description

The `cl_khr_mipmap_image_writes` extension adds OpenCL C built-in functions
to write to a mipmapped image.

If `cl_khr_mipmap_image_writes` is supported by the OpenCL device, the
`<<cl_khr_mipmap_image>>` extension must also be supported.
