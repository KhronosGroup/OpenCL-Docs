=== Description

{cl_khr_int64_base_atomics_EXT} adds built-in OpenCL functions supporting atomic
operations to be performed on 64-bit signed and unsigned integers in global
and local memory.

See the link:{OpenCLCSpecURL}#cl_khr_int64_base_atomics[64-Bit Base Atomics]
section of the OpenCL C specification for more information.
