=== Description

`cl_khr_int64_extended_atomics` adds built-in OpenCL functions supporting
extended atomic operations to be performed on 64-bit signed and unsigned
integers in global and local memory.

See the link:{OpenCLCSpecURL}#cl_khr_int64_extended_atomics[64-Bit Extended
Atomics] section of the OpenCL C specification for more information.
