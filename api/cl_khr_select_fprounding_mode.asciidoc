=== Description

`cl_khr_select_fprounding_mode` allows an application to specify the
rounding mode for an instruction or group of instructions in the OpenCL C
program source.

NOTE: This extension was deprecated in OpenCL 1.1, and its use is not
recommended.

See the link:{OpenCLCSpecURL}#cl_khr_select_fprounding_mode[Select
Floating-Point Rounding Mode] section of the OpenCL C specification for more
information.
