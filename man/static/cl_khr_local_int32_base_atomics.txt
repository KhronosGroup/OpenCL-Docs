= cl_khr_local_int32_base_atomics(3)

== Name

cl_khr_local_int32_base_atomics - Extension enabling base atomic operations on 32-bit integers to locations in __local memory.

== C Specification

[source,c]
----
#pragma OPENCL EXTENSION cl_khr_local_int32_base_atomics : enable
----

== Description

This extension was promoted to a core feature in OpenCL 1.1.
The built-in atomic function names were changed to use the `atomic_` prefix
instead of `atom_`.
