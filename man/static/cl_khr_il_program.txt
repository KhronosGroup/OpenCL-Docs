= cl_khr_il_program(3)

== Name

cl_khr_il_program - Enable loading SPIR IL programs

== C Specification

[source,c]
----
#pragma OPENCL EXTENSION cl_khr_il_program : enable
----

== Description

The OpenCL KHR extension reflink:cl_khr_il_program has been deprecated.
This feature is now core.
