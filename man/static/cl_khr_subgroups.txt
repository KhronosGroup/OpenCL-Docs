= cl_khr_subgroups(3)

== Name

cl_khr_subgroups - Deprecated extension

== C Specification

[source,c]
----
#pragma OPENCL EXTENSION cl_khr_subgroups : enable
----

== Description

The OpenCL 2.0 KHR extension `cl_khr_subgroups` has been deprecated.
The feature is now core.

== Document Notes

For more information, see the OpenCL Extension Specification at URL
