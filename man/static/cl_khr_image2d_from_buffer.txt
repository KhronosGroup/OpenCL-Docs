= cl_khr_image2d_from_buffer(3)

== Name

cl_khr_image2d_from_buffer - Extension enabling creating 2D image from buffer data

== C Specification

[source,c]
----
#pragma OPENCL EXTENSION cl_khr_image2d_from_buffer : enable
----

== Description

This extension was promoted to a core feature in OpenCL 2.0.

// == See Also

== Document Notes

For more information, see the OpenCL Extension Specification at URL
