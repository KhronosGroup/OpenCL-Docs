= Dead Links(3)

== Name

deadLinks - Dead Links From The OpenCL Reference Pages

== Description

If you've arrived at this page by following a link in the OpenCL 3.0
Reference Pages, the reason is that an automatically generated
cross-reference was created, even though the cross-reference target is not
part of the OpenCL 3.0 specifications. Details are explained individually
below for each such link.

:clext: https://www.khronos.org/registry/OpenCL/specs/3.0-unified/html/OpenCL_Ext.html
:clenv: https://www.khronos.org/registry/OpenCL/specs/3.0-unified/html/OpenCL_Env.html

[cols="1a,1a", options="header"]
|====
| Cross-Reference               | Why It Resolves Here
| `cles_khr_int64`              | See the link:{clenv}#_cles_khr_int64[OpenCL Environment specification]
| `get_image_num_mip_levels`    | See the
                                  link:{clext}#cl_khr_mipmap_image[OpenCL
                                  Extensions Specification]
|====
